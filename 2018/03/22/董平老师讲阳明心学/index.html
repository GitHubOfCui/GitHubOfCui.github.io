2018-03-22

    王阳明，也叫王守仁，是明朝中期的思想家，其开创的心学是儒学的一个分支，思想最早起源于孟子，少时对骑射颇感兴趣，不爱读书，兴趣极其广泛。王先生的主要思想有三点：心即理，知行合一，致良知。

王阳明的生平

  • 十二岁时王阳明问他的老师读书的目的是什么，老师回答是考取状元而入仕途，但王阳明认为读书是要做圣人，这正是儒家的思想精髓。
  • 十五岁时王阳明独自一人骑马去考察边关，研究兵法。
  • 二十八岁中进士。
  • 三十四岁因为得罪大官而被贬至贵阳龙场，在极其艰苦的条件下产生了历史上重要的龙场悟道。
  • 之后便是王阳明的思想逐渐完善的时间。

    心即理

    王阳明认为理并不是存在于某些特权阶级或圣人所独立的东西，不是需要我们去向他们求理，而是理就在人的心中。这比较像15世纪的西方宗教改革，新教认为上帝并不是有那些品行恶劣的教皇所掌握，而是应该在每一个人的心中，这导致了西方的文化启蒙运动。所以王阳明的心学是中国的思想启蒙。

    心即理是阳明心学的逻辑起点，这个心并不是我们常说的心脏，那只是一团血肉，王阳明所说的心是本心。阳明先生认为要成为圣人，最重要的是开显本心，天人合一。而去私语就是一个心明，理明，道明的过程，人的本质和天的本质是一致的，都包含宇宙间最高的真理即道。

知行合一

    王阳明认为的知行合一和前人的观点不太一样，如果仅从字面理解便是知道了道理便要去实践，这样理解虽然也无问题，但有些低估先生的思想了。我们经常说知道了许多道理却也过不好这一生，假如一个人只是在脑中知道了这个道理但是没有去做，那么我们能否认为这个人有了孝敬父母的认知呢，显然是不能的。所以王阳明认为知和行本来就是一回事，知之真切笃实处便是行，行之明觉精察处即是知，知行工夫本不可离。
    知是行的起点，行是知的完成，知和行本来就是一个过程的两端而已，而知识的最终形态是生活当中的实践，当我们谈论知时便已经有行在里面了(真知既是行)，当我们谈论行时也包括了知(知要通过行表现出来)。当我们想验证知是否正确时，要用到实际情况下验证其是否是真知。

格物致知

    格物致知是朱熹在理学中提出的概念，朱熹认为事物本身的理存在于事物本身，即具体事物之理，我们要不断的学习才能习得这个小理从而得到一般之理。

    王阳明的思想则与朱熹不同，先生把我和物的场景转化成我的实践过程，合乎本心，合乎天理。格物即正心，转化成了一个道德实践的命题。强调了我做为实践的主题在实践过程中的地位和作用。

致良知

    良知即本心，每一个人的良知都是光明通透的，不过会被欲所遮盖。所谓知之匪艰，行之惟艰就是因为良知的自知是容易的，只是致良知不是那么容易。良知本就是知善知恶知是知非的，是我们自己把“良知”盖住了而已，所以我们要通过反思把良知显露出来。所谓致良知可以看做两方面。

  • 一即反思即诚意：把我们的意识，观念，思想，念头等放在良知面前观照一番，合乎良知的叫做诚，不合的就是不诚，诚意即正心。
  • 二曰致吾心之良知于事事物物，就是明明德于天下。

私欲的来源问题

天道的本身呈现出一种善的状态，是无私的，事物从它们那里获取它的分殊之理，它的本质也是善的，同时我们的本心在它的最初状态也是有良知的，那么既然我们的本质是善的，所处的环境也是善的，那遮蔽我们良知的私欲是从哪来的？

天道是至善的，任何一种事物的原本状态也是好的。本质上来说，人之本质等同于天道，但人有物质的躯体，性是形而上者，我们还有一个形而下者–我们的躯体，它需要物质来滋养，所以人会产生一种本能的想法去获取物质来滋养我们的躯体。所谓欲望实际是本能的越界。

无心外之物

    阳明先生认为物既是事，事是介入了本体活动过程的。有一个很著名的事件是王阳明又一次带学生去山间赏花，学生问王阳明当你看到花的时候花在那里，当你没看到花的时候花在哪里？王阳明的回答是当你没看到花时，花和你的心同归于寂，当你看到花时，花在你心中一时便分明了起来。

    这里有一个很重要的概念寂，王阳明认为寂是事物存在的一种状态。客观事物有两种存在状态，一个是寂，一个是显。事物可以从寂瞬间转化到显的状态，事物可以同时处于这两种状态。这和现代物理学中的薛定谔的猫有异曲同工之妙，在你没看到这只猫的时候它处于一种状态，当你观察它的时候改变了它固有的状态。

    一件事物在成为我的对象之前，它的存在我不关心，对我来说没有意义，它不在我的主体世界之中。真正的仁人，是与天下万物为一体的，天下任何事情都和自己相关心。每个人都要致良知，使我们的良知显现出来，并且通过主体的实践尽可能的去扩展我们的心灵世界，使一切万物都能进入我们的主体世界，这样我们的心与天下一切事物相关，最后把天下所有事情都融入到自己主体世界当中那便是圣人，便是达到了无心外之物的境界。

    无心外之物还有另一层意思，王阳明担心的恰恰是心外有物，心外有物表明那个物与你没有关系，你的境界还不够。如果你只关心自己，那么你的世界只是一小块那么大，对于我们普通凡人，需要不断的通过实践把自我的主体世界和所谓的客观世界不断的链接起来，来扩充我们的主体世界，这样我们的生命境界才会越来越宏大，我们的意义世界才变得越来越宏大没主体世界也越来越宏大。

    无心外之物还有另一层理解，我的事它一定是和我相关的，凡是我的事都在我的心中，所以也可以理解外心外的事不是我的事便不是我的物，这个角度也可以诠释这句话。

四句教

  1. 无善无恶心之体
  2. 有善有恶意之动
  3. 知善知恶是良知
  4. 为善去恶是格物

下面对先生的这四句话进行一一阐明。

  1. 无善无恶是为至善，是本心自身实在的状态，它是未发之中，因为一切善和恶是经验世界当中的经验价值判断，它是绝对的价值原点。比如称，它没有重量所以才能去称重。比如眼睛里面没有颜色，所以才能分辨颜色，如果眼睛里面有颜色那便不能正常分辨颜色了。镜子中原本就没有东西，来了什么便照出什么，标准本身不该有任何东西存在，这就是本心。
  2. 意为心之动，为什么心要动，心要动是有原因的。良知从无善无恶的形而上转向了经验状态。对境或者物产生了动从而形成了意，要么就会落于善，要么落于恶，要么是无善无恶。这个过程可以理解成一种价值上的落实。
  3. 心体向意转化的过程中良知能判断哪个是善哪个是恶，良知本就能辩善恶。
  4. 当良知做出了关于善恶的判断之后，我们要听从良知的召唤去做，善小但不去做是知行不合一，明知是恶还去做也是知行不合一。
